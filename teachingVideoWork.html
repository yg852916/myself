( Unity 開發 )自製 遊戲教學影片一 持續更新中

影片說明：

在開發遊戲中，順便拍下影片記錄過程，順便給一些有興趣的人學習!!
主要是第三人稱遊戲槍戰內容。
Unity Multiple Game 01 (角色建立、移動)
Unity Multiple Game 02 (跳躍、第三人稱視角)
Unity Multiple Game 03 (不使用character controller，進行跳躍移動 )
Unity Multiple Game 04 (Animator Rigging 控制握槍姿勢)
Unity Multiple Game 05 (撿槍與丟槍 使用Raycast方式)
Unity Multiple Game 06 (控制槍、上半身的瞄準，使用multi-Aim-constraint、multi-position-constraint)
